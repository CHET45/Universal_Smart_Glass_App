# HeyCyan Core Modules

This directory contains the core libraries for HeyCyan Smart Glasses functionality.

## Modules

- **core-audio** - Audio capture and processing
- **core-ble** - Bluetooth Low Energy communication with glasses
- **core-connectivity** - Network and connectivity utilities
- **core-data** - Data models and storage
- **core-summarization-api** - AI summarization interfaces
- **core-transcription-api** - Speech-to-text interfaces
- **core-utils** - Shared utilities

## Building

These modules are included as a Gradle composite build. When you build CyanBridge, these modules are automatically compiled and included.
